# Polychromatic theme

A simple light and dark colour theme and product icon theme for VS Code.

Only tested on Arch Linux (blur effect from Hyprland).

Some icons are not mapped yet and there are a few colours that need changing.

![light](assets/light-2.0.0.png)

### Credit
- Product icon theme uses [Material Symbols](https://fonts.google.com/icons).
- Colour theme uses [Custom UI Style](https://marketplace.visualstudio.com/items?itemName=subframe7536.custom-ui-style) extension for transparency.
- Original colours for dark theme from [Material Theme](https://github.com/t3dotgg/vsc-material-but-i-wont-sue-you) (specifically Material Theme Darker).
